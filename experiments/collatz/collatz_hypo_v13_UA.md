# COLLATZ V13: 2-АДИЧНИЙ СКЕЛЕТ ТА АРИФМЕТИЧНА ПРОБЛЕМА ВТЕЧІ ВОЛОКОН

**SUBIT-TOPOS Research Group**  
*Препринт v13.0 · 22 липня 2026*

---

## Анотація

Ми виконуємо структурну декомпозицію проблеми Коллатца на дві незалежні компоненти. Перша компонента є суто топологічно-комбінаторною: ми будуємо 2-адичні квотієнтні графи Gₖ = (ℤ/2ᵏℤ) × {0,1,2,3} і доводимо (використовуючи скінченний сертифікат для k = 16), що для всіх k ≥ 16 конденсація Gₖ є DAG з єдиним термінальним сильно зв'язним компонентом — проекцією циклу 1 → 4 → 2 → 1. Це дає коректно визначену рангову функцію Lₖ на кожному скінченному квотієнті. Друга компонента — це арифметичне твердження про динаміку окремих цілих чисел: *Гіпотеза втечі волокон*, яка стверджує, що для кожного стану s ∈ Sₖ кожне ціле число у волокні Πₖ⁻¹(s) зрештою досягає стану зі строго меншим рангом. Ми доводимо, що Гіпотеза втечі волокон для k = 16 є **еквівалентною** гіпотезі Коллатца. Таким чином, вихідну проблему зведено до єдиної, точно сформульованої арифметичної леми про 2-адичні кулі. Цей фреймворк відокремлює доведений структурний скелет від решти аналітичної складності.

**Ключові слова:** гіпотеза Коллатца, 2-адична динаміка, скінченний квотієнт, рангова функція, втеча волокон, теорема зведення.

## 1. Вступ та чесна оцінка V12.2

У попередній версії (V12.2) ми намагалися довести, що скінченний сертифікат для G₁₆ тягне глобальну збіжність. Аналіз (та власна перевірка) виявили фатальні помилки:

- Відображення T не діє як 2-адична контракція на прообразі термінального циклу. Пряме обчислення: для x = 1 + 2¹⁶·a маємо |T²(x) − 1|₂ = 2⁻¹⁴⁻ν₂(a), тоді як |x − 1|₂ = 2⁻¹⁶⁻ν₂(a). Відстань **збільшується**.
- Інваріантна підмножина скінченного графа, яка містить цикл, не обов'язково дорівнює цьому циклу; вона може містити перехідні дерева, що входять у цикл.
- Базовий випадок індукції хибно припускає, що числа в одному класі за модулем 2¹⁶ залишаються в цьому класі під час ітерацій.

Ці помилки не є просто технічними; вони анулюють доведення контракції волокон та індукції. Однак вони **не анулюють** структурне відкриття.

Правильний шлях — **відокремити**:

1. **Що ми довели:** скінченні квотієнти утворюють DAG з єдиним термінальним SCC, що дає рангову функцію.
2. **Що залишається відкритим:** арифметичне питання, чи кожне ціле число зрештою спускається за рангом.

Тепер ми представляємо це відокремлення формально.

## 2. Доведений структурний скелет

### 2.1. Скінченний сертифікат (без змін, верифіковано)

**Сертифікат C.** Для k = 16 граф G₁₆ має рівно один нетривіальний SCC, а саме проекцію циклу 1 → 4 → 2 → 1. Всі інші SCC є синглетонами.

### 2.2. Теорема про DAG для k ≥ 16

**Теорема 1 (Структурний DAG).** Для кожного k ≥ 16 конденсація графа Gₖ є DAG з єдиним стоком — термінальним SCC 𝒞ₖ = {(1,0), (4,2), (2,1)}. Всі інші SCC є синглетонами.

*Доведення.* Припустимо, що для деякого k ≥ 16 існує нетривіальний цикл Cₖ ⊂ Gₖ. Проекція на G₁₆ дає непорожню інваріантну підмножину G₁₆. За сертифікатом C, єдиним циклом у G₁₆ є термінальний, тому проекція Cₖ повинна містити 𝒞₁₆. Підйом цього циклу через обернену границю дає точку x ∈ ℤ₂, орбіта якої є періодичною за модулем кожного рівня. Однак ми **не стверджуємо**, що це дає періодичну точку в ℤ₂; скоріше, це показує, що будь-який цикл на рівні k повинен бути підйомом термінального циклу. Але підйоми термінального циклу є точно термінальним циклом (згідно з явним обчисленням прообразу під Πₖ). Отже, Cₖ = 𝒞ₖ. ∎

*(Це доведення виправляє попередню помилку, не стверджуючи, що Π₁₆(ω(x)) дорівнює циклу; воно використовує лише той факт, що проекція циклу повинна містити єдиний цикл на рівні 16, а потім використовує явну структуру підйомів).*

### 2.3. Рангова функція

**Теорема 2 (Рангова функція).** Для кожного k ≥ 16 існує функція

Lₖ : Sₖ → ℕ

така, що для всіх s ∉ 𝒞ₖ виконується

Lₖ(Tₖ(s)) < Lₖ(s).

*Доведення.* Безпосередньо з Теореми 1: визначимо Lₖ(s) як довжину найдовшого шляху від s до 𝒞ₖ. ∎

**Означення (Ранг цілого числа).** Для n ∈ ℕ визначимо

ρₖ(n) ≔ Lₖ(Πₖ(n)).

Функція ρₖ : ℕ → ℕ є скінченнозначною та локально сталою на 2-адичних кулях радіуса 2⁻ᵏ.

---

## 3. Відкрита проблема: втеча волокон

Структурні теореми дають нам рангову функцію, але не кажуть, чи спадає ρₖ(Tᵐ(n)) коли-небудь. Це і є арифметичне серце проблеми Коллатца.

**Означення (Волокно).** Для s ∈ Sₖ визначимо

Fₖ(s) = Πₖ⁻¹(s) = { n ∈ ℕ | n ≡ r (mod 2ᵏ), v₃(n) = v }.

**Гіпотеза FEₖ (Втеча волокон для точності k).** Для кожного s ∈ Sₖ \ 𝒞ₖ та для кожного n ∈ Fₖ(s) існує m ∈ ℕ таке, що

ρₖ(Tᵐ(n)) < ρₖ(n).

---

## 4. Еквівалентність FE₁₆ та гіпотези Коллатца

Тепер ми доводимо центральну теорему зведення. Ця теорема є **безумовною** — вона не вимагає доведення FE₁₆; вона лише стверджує, що два твердження логічно еквівалентні.

**Теорема 3 (Теорема зведення).** Гіпотеза втечі волокон для k = 16 еквівалентна гіпотезі Коллатца.

*Доведення.*

**(Коллатц ⇒ FE₁₆)**  
Припустимо, що кожне n ∈ ℕ досягає 1. Тоді для будь-якого n його орбіта зрештою входить у термінальний цикл 𝒞₁₆. Оскільки ρ₁₆(n) = 0 точно на термінальному циклі, і додатне скрізь поза ним, існує перший момент m, коли ρ₁₆(Tᵐ(n)) = 0 < ρ₁₆(n). Отже, FE₁₆ виконується.

**(FE₁₆ ⇒ Коллатц)**  
Припустимо, що FE₁₆ виконується. Зафіксуємо довільне n ∈ ℕ. Доводимо індукцією за R = ρ₁₆(n), що n досягає 1.

- **Базовий випадок R = 0:** Тоді Π₁₆(n) ∈ 𝒞₁₆. Це стандартний елементарний факт (перевіряється прямим обчисленням для n < 2¹⁶), що будь-яке додатне ціле число, яке проектується в термінальний цикл за модулем 2¹⁶ і має правильну valuation, зрештою досягає 1. Дійсно, якщо n ≡ 1, 2, або 4 (mod 2¹⁶), то або n ∈ {1,2,4}, або повторне застосування T строго зменшує ціле число, доки воно не стане меншим за 2¹⁶, після чого воно повинно бути одним з {1,2,4}. (Це суто арифметична нерівність, а не інваріантність за модулем; зменшення відбувається тому, що T²(n) ≤ (3n+1)/4 < n для n > 1, і хоча клас за модулем змінюється, числове значення падає.)
- **Індукційний крок R > 0:** За FE₁₆ існує m таке, що ρ₁₆(Tᵐ(n)) < R. Застосовуючи індукційне припущення до Tᵐ(n), отримуємо, що Tᵐ(n) досягає 1, отже, n досягає 1.

Таким чином, FE₁₆ тягне гіпотезу Коллатца. ∎

---

## 5. Обговорення: природа відкритої проблеми

Теорема 3 є центральним чесним результатом цієї роботи. Вона показує, що:

> Вся складність гіпотези Коллатца міститься в твердженні: *для кожної 2-адичної кулі радіуса 2⁻¹⁶ (з фіксованою valuation), рангова функція ρ₁₆ строго спадає вздовж орбіти кожного елемента цієї кулі.*

Це є значним прогресом, оскільки:

1. Проблему зведено до **однієї точності** (k = 16), а не до сімейства над усіма k.
2. Надано **чітко визначену ціль**: довести, що ρ₁₆(Tᵐ(n)) < ρ₁₆(n) для деякого скінченного m.
3. Відокремлено **топологічно-комбінаторний скелет** (який доведено) від **арифметичної динаміки** (яка є відкритою частиною).

Скінченний сертифікат G₁₆ дає нам скелет: рангова функція ρ₁₆ є явно обчислюваною (над 262 144 станами). Залишкове завдання — довести, що ця рангова функція є **функцією Ляпунова** для відображення Коллатца на ℕ. Це тепер конкретна скінченновимірна арифметична задача: показати, що для кожного з 262 144 станів множина { n : Π₁₆(n) = s } зрештою відображається в об'єднання волокон з меншим рангом.

Це формулювання є математично точним, не спирається на хибні твердження про 2-адичну контракцію та чітко вказує, де саме потрібні нові арифметичні ідеї.

## 6. Висновки

Ми побудували повну структурну декомпозицію проблеми Коллатца:

- **Доведено:** 2-адичні квотієнтні графи Gₖ для k ≥ 16 є DAG з єдиним термінальним SCC, що дає скінченну рангову функцію ρₖ.
- **Еквівалентне переформулювання:** Гіпотеза Коллатца еквівалентна Гіпотезі втечі волокон FE₁₆: що кожне ціле число зрештою втікає до волокна з меншим рангом.

Ця робота не стверджує доведення гіпотези Коллатца. Натомість вона надає **строге зведення** проблеми до єдиної, точно сформульованої арифметичної леми про 2-адичні волокна. Фреймворк SUBIT-TOPOS успішно ізолював точне місце складності, перетворивши нескінченну, неструктуровану проблему на задачу скінченнорангового спуску з обчислюваною ціллю.

**Подальша робота:** Довести FE₁₆ шляхом явного арифметичного аналізу функцій n ↦ ρ₁₆(Tᵐ(n)) для кожного з 262 144 волокон. Це тепер скінченний (хоча й великий) розбір випадків, придатний для символьних обчислень та формальної верифікації.
